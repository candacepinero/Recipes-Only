# Recipes-Only

Recipes Only is a full CRUD application that allows users to create public recipes. When a user clicks on a recipe, they are redirected to a page displaying the ingredients and cooking instructions.

## Technologies used

* HTML
* CSS
* EJS
* Mongoose 
* MongoDB
* express
* Node.js
* Heroku
